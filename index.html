Thousands of reactive channels on a single persistent connection.

Channels are a fundamental concept in Theron. Channels are a flexible way to broadcast data across segments of your users. Theron utilizes a Pub/Sub pattern for data streaming, which lets you push data to global audiences instantly.

Theron is built on top of ReactiveX: a de facto standard paradigm for responsive, resilient, elastic, and message driven systems. Last but not least, Theron fits nicely with the modern frameworks such as React and Angular.

Smart realtime streaming with the most advanced database.

Databases are the core of any application. Theron connects directly to the Postgres alongside your server and uses all of the advantages of SQL databases. Put simply, you can write plain queries to fetch, order, and stream data.

Depending on the query to the database, Theron understands which data is important and should be distributed, so it sends only payloads: the small artefacts of the query result needed in order to construct an entire dataset.
